## 项目介绍

基于[玩Android API](https://www.wanandroid.com/blog/show/2) 使用vue.js开发的客户端

## 用到的三方库

- UI控件主要使用的是[Vant](https://youzan.github.io/vant/?source=vuejsorg#/zh-CN/)
- 网络访问使用的是[axios](https://github.com/axios/axios)
